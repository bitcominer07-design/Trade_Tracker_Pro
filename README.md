# Trade_Tracker_Pro

Trade_Tracker_Pro is a desktop application for tracking and journaling your trades.  
Keep a clean record of your trades, analyze your performance, and organize your trading history easily.

---

## Features
- Log and review trades quickly
- Export your trading history
- Clean and user-friendly interface
- Windows installer included

## Installation Instructions

1. Download the `.zip` or `.exe` from the release link above.  
2. If `.zip`, extract it to a folder of your choice.  
3. Run `Trade_Tracker_Pro.exe` to launch the app.  
4. (Optional) Create a desktop shortcut during installation.
5. (Optional) install libreooffice https://ninite.com/libreoffice/
